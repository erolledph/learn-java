# ☕ Java Learning Hub

Learn Java with an interactive AI tutor - **Java runs directly in your browser!**

## Quick Start

1. **Open `index.html`** in your browser
2. Click **"Start Learning Java!"**
3. Start coding!

## Features

### 🤖 AI Tutor (Byte)
- Explains Java concepts
- Helps debug your code
- Interactive conversations
- Provides examples and challenges

### 💻 Run Java in Browser
- Click **▶️ Run** to execute Java code
- No downloads or installations needed!
- Works directly in your browser

### 📚 55 Interactive Lessons
- Progressive difficulty
- AI-powered explanations
- Practice exercises
- Progress tracking

## How Java Execution Works

When you click **Run**:
1. Your code is sent to an online compiler
2. Java code is compiled and executed
3. Output is displayed in the browser

**No Java installation required!** All Java code runs in your browser.

## Running Java Code

### Direct Browser Execution (Default)
1. Write your Java code
2. Click the **▶️ Run** button
3. See output instantly!

### Using Local JDK (Optional)
If you want to use your local Java installation:

```bash
cd C:\Users\rebec\Downloads\Java
node java-runner.js
```

Then click Run in the browser for instant execution.

## Troubleshooting

### Java won't run
- Check your internet connection (required for online compiler)
- Try refreshing the page
- Make sure your code has no syntax errors

### AI not responding
- Check your internet connection
- The API key should be pre-configured
